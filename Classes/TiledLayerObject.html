TiledLayerObject

open class TiledLayerObject: SKNode, SKTiledObject

The TiledLayerObject is the base class for all SKTiled layer types.

This class doesn’t define any object or child types, but manages several important aspects of your scene:

  • validating coordinates
  • positioning and alignment
  • coordinate transformations
